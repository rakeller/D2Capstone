Weekly Blog Post #2

Team Focus

Posted by Team D2 on March 22, 2018

Group Progress

Worked together on figuring out how individual parts will come together via Java, Android Studio, C++, and Bluetooth. Started with how Android phone will get data from hardware in order to make the app work properly.

Individual Progress

Robert Keller: Continued working on the analysis section of the phase vocoder and has started on the synthesis section. Continuing in Matlab and will use Simulink to convert filters and waves into C++ functions.

Julian Trulear: Finished soldering pins on Bluefruit LE and has also finished soldering pins on the LSM9DS0 (gyroscope). Started routing tiny wires from Bluefruit to LSM9DS0 for correct power and data transfer. Tiny wires because want as minimal obstructions as possible for hardware.

David Yuan: Created a simple UI for the application as we want it to not be distracting for the user. Focusing more on the functionality and processing inside the app, e.g. how the modes will work in Android programming.

Goals for Next Week

Want analysis section of the phase vocoder to be done as well as a decent chunk of the synthesis section. Want all wires to be hooked up correctly and data to be transferred in hardware. Want UI to be functional on the phone and to start transferring data between the phone and hardware.
